## 📌 Project Overview

Employee Salary Prediction is a Machine Learning project developed to predict an employee's salary based on various factors such as age, gender, education level, job title, and years of experience.

The project follows a complete Machine Learning workflow including data preprocessing, feature encoding, model training, model evaluation, and deployment using Streamlit Cloud.

An interactive web application has been developed using Streamlit, allowing users to enter employee details and instantly receive salary predictions.



## 🎯 Problem Statement

Organizations often need to estimate employee salaries based on different professional and demographic factors.

Manual salary estimation can be inconsistent and time-consuming.

This project aims to build a Machine Learning model that can accurately predict employee salaries using historical employee data.



## 🎯 Objectives

- Build a salary prediction model using Machine Learning.
- Perform data preprocessing and feature engineering.
- Train and evaluate predictive models.
- Create an interactive Streamlit application.
- Deploy the application for public access.
- Provide real-time salary predictions through a user-friendly interface.
